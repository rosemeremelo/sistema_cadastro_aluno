# Sistema de Cadastro de Alunos 🎓

Projeto desenvolvido para a disciplina de **Qualidade de Software**. O sistema permite o gerenciamento (CRUD) de alunos em memória local.

## 🚀 Funcionalidades
- Cadastro de alunos (Nome, Matrícula, E-mail, Turma).
- Validação de campos obrigatórios e matrículas duplicadas.
- Listagem dinâmica em tabela.
- Remoção de registros com confirmação.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3 (com foco em responsividade e acessibilidade)
- JavaScript Vanilla (Lógica e manipulação de DOM)

## 📊 Métricas de Qualidade
Este projeto foi documentado utilizando a **Análise de Pontos de Função (APF)**, obtendo uma Média Quadrática de complexidade de **4,35**.

1. Clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
